## üöÄ Strategy for "Number 1 Web App in the World"

### 1. The "Thinker-Doer" Architecture
Use **Thinking Mode** for the high-level script writing and storyboard planning, and **Flash-Lite** for the instant UI feedback (the "HUD"). This makes the studio feel both intelligent and incredibly snappy.

### 2. Multi-Modal Consistency
Combine **Nano Banana Pro** with **Veo 3** to create "Reference-Bound" generation. An image generated in the concept phase should serve as the anchor for the video generation to ensure the character doesn't change between shots.

### 3. Voice-First Directing
Implement **Gemini Live** as a "Director's Assistant." Instead of clicking nodes, the user should be able to say, *"Hey Gemini, add a dramatic low-angle shot with neon lighting to the second scene,"* and have the UI react in real-time.

### 4. Real-World Grounding
Use the **Google Search & Maps** integration to allow creators to ground their fictional worlds in reality. Finding a "desert location at sunset" should pull real coordinate lighting data to influence the AI's lighting model.
